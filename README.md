
# GlobalSolution1
  Assistente Virtual de Bem-Estar no Trabalho
Nome: "MindWork" 

✅ Saúde mental é prioridade pós-pandemia
✅ ODS 3 (Saúde e bem-estar) + ODS 8
✅ Tema sensível e humanizado

MVP Funcional:
Chatbot interativo (JS puro com respostas programadas)
Check-in diário de humor com gráfico de evolução
Biblioteca de técnicas (respiração, pausas, mindfulness)
Alertas de pausa personalizáveis
Dashboard de métricas de bem-estar ao longo do tempo
Recursos de emergência (links para apoio profissional)

Diferencial técnico:
Chatbot com árvore de decisão em JS
Notificações do navegador (Notification API)
Gráficos de humor com Chart.js ou CSS
LocalStorage para histórico

Fazer com que o usuário entre todos os dias no site, com o intuito dele colocar uma nota de 1 a 5 demostrando como ele esta se sentindo, isso vai gerar uma avaliação mensal da sua situação emocional no trabalho. Caso esteja ruim iremos sujerir tecnicas( ex - inspira 5s, segura5s, respira5) de como melhorar isso e direcionar o contato de expecialistas.

Tambem teria a opção de colocar o metodo pomodoro, com o ajuste das horas e pausas de acordo com o usuario.

Demonstrar um grafico numa tela, aonde monstra a situação da saude mental do usuario de acordo com as notas de 1 a 5 dadas poe ele.
